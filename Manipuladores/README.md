# Laboratorio de Robótica (4º GIERM)

Sesión 7/10

En esta sesión hemos trabajado con el robot ABB usando los programas ya realizados en la sesión anterior. La velocidad decidida para el robot soldador ha sido v10, ya que la velocidad promedio de un robot soldador para asegurar que suelda bien es de 60 cm/min, es decir de unos 10 mm/s. Al simular el programa el ABB ha realizado el recorrido de forma satisfactoria.

Sesión 21/10 
En esta sesión hemos creado el modelo físico de nuestro efector final, convirtiendolo en una herramienta y acoplandolo al ABB en el entorno de simulación. Hemos empezado a investigar la forma para agarrar y soltar objetos con la propia herramienta, y el objetivo de la siguiente sesión será integrarlo en nuestro efector.
